The Smart Cookies Sensor Ecosystem is a highly integrated mesh network of wireless sensor nodes capable of long-term datalogging and realtime telemetry visualization. Built with students in developing communities in mind, it is completely self-contained and self-working, requiring no programming, software downloads, or even Internet access to work.

While an individual Smart Cookie can act as a standalone data collection tool, multiple Smart Cookies within close proximity of each other will self-organize to form a decentralized, distributed sensing mesh network. When powered on, each Smart Cookie activates a local WiFi network which, when accessed by any device with a browser, displays a dashboard interface which allows users to visualize realtime telemetry and data analytics from all Smart Cookies connected to the mesh network.

Envisioned Applications

Smart Cookies provide exciting opportunities for educational technology initiatives in remote areas, university research, digital agriculture, smart city planning, and any industry with distributed sensing needs.

Engineering

Each Smart Cookie is comprised of an ESP8266 WiFi module, rechargeable battery, and onboard sensor array.

Embedded Web App

A core requirement is the ability to work completely offline, with no software downloads requiring Internet connectivity required. the decreasing cost of generic Android tablets Functional web app should be stored completely within the onboard flash memory of the ESP82666 module, while still allowing room for the program. Prototypes of the embedded web app use Google's Material Design Lite framework to achieve a minimal, clean, and functional user interface. Chart.js is used for the data visualization elements and smooth animations that add vitality to the realtime data stream.


Mesh Networking

While a single Smart Cookie can be used as a self-contained data collection tool, using multiple Smart Cookies in close proximity with each other automatically creates a decentralized mesh network that intelligently routes data packets through other Smart Cookies through a root node to which a low-cost tablet can be connected to visualize realtime telemetry from all Smart Cookies. This allows the effective range of data collection extend far beyond the estimated 100m signal radius of a single WiFi module.
