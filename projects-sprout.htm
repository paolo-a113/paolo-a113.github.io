Engineering

Automatic OTA Firmware Updates

Sprout automatically updates its firmware when it connects to the Internet. This is accomplished through a remote Amazon EC2 instance running a Node-RED application.


Persistent Configuration

Users can change the settings of any node in the mesh network through a web app. This is accomplished with a back-and-forth exchange of JSON packets, addressed to any or all of the connected nodes. Any configurations that need to be preserved for the next startup (such as node names) are saved to each node's flash memory and are recalled from memory upon boot.
