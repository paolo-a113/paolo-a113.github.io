The Easiest Way to Plan a Trip

Whether I'm making a solo trip itinerary or planning a trip with friends and family, the workflow is usually the same: read some travel blogs to find must-see places, use Google Maps for research, then switch to Google Docs or Microsoft Word to actually create the itinerary. While building an itinerary in a text editor is simple, lightweight, flexible, customizable, and easily shareable, it's also very prone to typos and errors. And if plans change that affect the timing of other events, it's a hassle to have to go through the entire itinerary to make the necessary changes.

Passportable is a travel planning platform that makes it easy for anyone to create and share itineraries. Places and activities become "cards" which users can add to their itinerary and drag to reorder. Users can specify how long they'll be staying at each location for. Passportable automatically updates the itinerary with transportation information and suggested leave times. with a clean, responsive user interface inspired by retro travel postcards. If you're going on a trip with friends, you'll have the option of sharing your itinerary with them so that everyone is on the same page.7

Technology

Under the hood, Passportable is a simple web app with a frontend built with HTML5, CSS3, and Javascript (with some JQuery and Bootstrap) and a Firebase backend. It relies on the Google Maps and Mapbox APIs for its navigation features.


Commercialization

While a version of this platform will be made freely available to anyone who wishes to use it, it will generate revenue with ''freemium'' pricing.
